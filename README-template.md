## Getting Started

Ces instructions simples permettent de déployer le serveur Express d'Endagri

### Installation

Clôner le répertoire

```
git clone
```

Installer les dépendances

```
cd endagri-server
npm install
```

## Développement

Lancer l'environnement de développement

```
npm run dev
```

## Production

```
npm run build
npm run start
```

## Lancer les tests (pas mis à jour)

```
npm run test
```
